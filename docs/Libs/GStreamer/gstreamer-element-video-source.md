# \[GStreamer\] Video Source

## Video Test Source

Chạy một __Video Test Source__ bằng lệnh sau:

```text
gst-launch-1.0 videotestsrc ! autovideosink
```

[Tài liệu tham khảo ở đây!](https://gstreamer.freedesktop.org/documentation/videotestsrc/?gi-language=c)
