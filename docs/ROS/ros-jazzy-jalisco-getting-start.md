# \[Jazzy Jalisco\]<br>Getting Start

## Tạo Môi Trường

### Khởi động môi trường

!!! danger "Nhớ"
    Mỗi lần khởi động đều phải chạy cấu hình này.

=== "Linux + macOS"
    ```bash
    source /opt/ros/jazzy/setup.bash
    ```
    Hoặc
    ```bash
    . /opt/ros/jazzy/setup.bash
    ```
=== "Windows"
    ```cmd
    call C:\dev\ros2\local_setup.bat
    ```
