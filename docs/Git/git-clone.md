#  Git Clone

!!! note "Note"
    Lệnh này có tác dụng kéo source code từ kho lưu trữ git về máy của bạn.

Thông thường chúng ta sẽ sử dụng lệnh đơn giản:

## Mô tả

Lệnh này sẽ kéo về tất cả các thứ từ kho lưu trữ về máy, nó bao gồm cả 

```bash
git clone $LINK
```


```bash
git clone --depth $LEVEL $LINK
```

```bash
git clone --depth 1 $LINK
```
