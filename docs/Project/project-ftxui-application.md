# Ftxui Application

**Ftxui Application** là dự án dùng để tạo các _**ứng dụng terminal đa nền tảng**_. Ứng dụng được xây dựng qua cmake và g++, khá cơ bản.

Trong dự án này mình đã đóng gói sẵn các tệp dựng sẵn (`lib*.a`) rồi và chỉ cần dựng thôi, tốc độ sẽ là nhanh hơn bình thường nhiều.

Nhờ dự án này mình dần hiểu tại sao các dự án lớn đa hệ điều hành lại cần có cách tổ chức khó khăn đến vậy.

Tải về dự án với đường dẫn sau:

=== "SSH"
    ```text
    git clone git@github.com:dothanhdathp/ftxui-application.git
    ```
=== "HTTPS"
    ```text
    git clone https://github.com/dothanhdathp/ftxui-application.git
    ```
