# Audio WAV FILE

## Định Nghĩa

- **.wav** là định dạng âm thanh tiêu chuẩn, được phát triển bởi công ty **Microsoft**.
- Các tệp `wav` có cấu trúc đơn giản nhằm hỗ trợ phát lại các tệp âm thanh tiêu chuẩn, nó không kèm các tệp _metadata_ bổ sung như tác giả, ảnh bìa, ...
