# Android Hardware abstraction layer (HAL)

_A hardware abstraction layer (HAL)_ là loại lớp trừu tượng có giao diện chuẩn để các nhà cung cấp phần cứng triển khai. HAL cho phép các nhà cung cấp phần cứng triển khai các tính năng cấp thấp hơn, dành riêng cho thiết bị mà không ảnh hưởng hoặc sửa đổi mã ở các lớp cấp cao hơn.
