# \[Win\] Command Prompt

__CMD__, hay __Command Prompt__, là một ứng dụng dòng lệnh trong hệ điều hành Windows cho phép người dùng tương tác với máy tính bằng cách nhập các lệnh văn bản. Công cụ này hữu ích để thực hiện các tác vụ quản trị hệ thống, quản lý tệp, kiểm tra kết nối mạng và khắc phục sự cố nhanh chóng mà không cần dùng giao diện đồ họa phức tạp.
