# Linux Bash

__Linux Bash__ là shell cơ bản của Linux, cùng với __sh__

## Giới thiệu

Đại đại thôi chứ nó cũng không có gì cả.

## Tính Năng

Thuộc phạm vi _shell-script_ thế nên các chức năng của Shell phải tuân thủ [POSIX Shell](/OS/os-posix-shell/).

Ngoài ra bash còn mở rộng thêm khá là nhiều cơ chế mới về xử lý mảng, chuỗi, vòng lặp, ... So sánh thì sh chỉ tiện viết các kịch bản hệ thống trên tệp tĩnh thì bash giúp việc đó làm dễ dàng hơn nhiều trên command, tối ưu triệt để thao tác với cửa sổ. bash tạo thành một khung hoàn chỉnh để viết các lệnh phức tạp, thuận tiện để viết những __*kịch bản dễ sửa đổi*__ giúp ngành lập trình phát triển mạnh.
