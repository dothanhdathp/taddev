# Zorin Font

## System Fonts

- Font chữ tải về để tại thư mục `/usr/share/fonts`. 
- Thư mục này thường được chia thành các thư mục con như **truetype**, **opentype**, hoặc **conf.d**. Bạn cần quyền _**root (sudo)**_ để thêm hoặc xóa tệp tại đây.

## User Fonts

- Font chữ tải về để tại thư mục `~/.local/share/fonts`.
- Các _fonts_ chữ tại đây chỉ khả thi trong khu vực không gian tên người dùng. Tức là nếu sử dụng một tài khoản khác đăng nhập vào sẽ không thể truy cập được các tệp phông chữ.
