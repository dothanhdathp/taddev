# Linux System Folder

## Font

Cài đặt font cho Linux cũng khá đơn giản. Tải về các tệp `.ttf` từ bất cứ nguồn nào. Chẳng hạn [Google Fonts](https://fonts.google.com/) hoặc [DaFont](https://www.dafont.com/).

Sau khi tải về sao chép vào các thư mục:

- `/usr/share/fonts/`: font chữ toàn hệ thống.
- `~/.local/share/fonts/`: font chữ cá nhân người dùng.

Cách khác là sử dụng công cụ ngoài như __Font Viewer__ hoặc __Font Manager__, cơ mà có lẽ chả cần đâu.
