# File

Hệ thống tệp trong Ubuntu rất đơn giản.

## Loại Tệp

- Đầu tiên cần làm quen với lệnh `ls -la`. Lệnh này sẽ cho xem toàn bộ danh sách tệp hiện có.
    - [Tham khảo thêm về ls](./linux-tools-ls.md)
- Có hai loại tệp cơ bản là `d` và `-`
    - `d` tức là _diractory_, nghĩa là đường dẫn hoặc là **thư mục** như cách hiểu của Windows
    - `-` là các tệp nhị phân _(binary)_. Các tệp này có thể đọc ghi hoặc thực thi.
